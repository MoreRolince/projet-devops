# Projet DevOps – Python


## Objectif du projet

L’objectif de ce projet est de transformer une base Python simple en un projet
**industrialisable**, en appliquant les pratiques DevOps vues en cours et en TP :

- structuration propre du repository,
- implémentation de logique métier testable,
- tests unitaires automatisés,
- packaging Python standard,
- intégration continue avec GitHub Actions,
- gouvernance Git sur la branche principale.
